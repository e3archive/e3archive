How To Backup ?

  1. Click Here to access the source code of this project.
  2. Click on the green colored button saying Code and Download ZIP.
  3. Extract the ZIP file and open index.html, Now you can navigate around as usual.
  4. Voila ! your backup is there. Although some functionalities may not work when offline. Remember that the source code of this website does not contain the files (JEE Material).

Downloading Files

  1. Go to Files section of this website.
  2. Find the file which you need to Download, you will get two links. First one is hosted using Send. Second link is hosted on Lufi.
  3. The First link has to be renewed every 7 days, and it cannot be taken down. The Second link has to be renewed every 45 days, and it can be taken down.
  4. There are some permanent file hosting services too but they can take down the material any time they want.
  5. The First link will be updated every Sunday, and the Second link will be updated every 30 days. Make sure to remind me if I forget. ;)
