Downloading Files

  1. First of all, Download Tor Browser.
  2. Find the file which you need to Download, you will get a link ending with .onion, and a Private Key. Paste that Link in Tor Browser. You will need the Private Key later.
  3. After doing the step above, You Will be prompted to enter a Private Key. Enter the key which was mentioned in the earlier step.
  4. You will be redirected to a page, Where you can download your file.

Managing Archive

  • First of all access the source code from here
  • Just in case, The archive goes down or stays inactive for long time. You can host the website on your own.
  • I expect the person managing this archive has a basic knowledge of HTML.
  • Let me tell you how to add a new entry in the archive. Let the name of entry be- xyz material.
  • First of all, upload the file on Tor using OnionShare.
  • After uploading it there, you will get a link ending with .onionand a Private Key. You will need these two things to allow people to download the files.
  • Now go to the filesFOLDER and make a new file there named xyzmaterial.html.
  • Edit the xyzmaterial.htmlby taking source code from other .html files in the filesfolder.
  • Make sure to add the .onion link and the Private Key.
  • Now go to files.htmland make a new entry there. I already mentioned that the user should know basics of html if he/she is managing the archive. I won't teach you HOW TO write html code. I expect you to know that.
  • Now go to updates.htmland make a new entry there.
  • Now go to search.htmland make a new entry there and don't forget to hrefit to files/xyzmaterial.html.

Why Tor Network ?

This website is NOT the kind of website which you find on Internet, When you search for pdf'sof various books or coaching modules.

It's an archive, It's purpose is to conserve the history of this exam. The materials used for preparing for this exam in different eras whether it's the 90's or after 2007(major change in the pattern). This archive contains almost all the JEE Previous Year Papers, Modules of iconic Kota Coaching Industry, Test papers of all difficulties, Famous books of all the eras (relevant to JEE), etc.

Ok, Now let's come to the point. I hosted this archive on Tor for two reasons

  • To prevent copyright infrigment and to prevent it from getting copyrighted.
  • To make the download process of files much difficult, I do not want this website to get potrayed as a hub for downloading copyrighted materials. I do not want this website to look illegal because my intentions are to make this an archive website, not like various telegram channels where you find various JEE material. Their intentions are not noble and what they are doing is illegal. Opposite of that, My intentions are good and I want to setup an archive, where anyone from any era can see the beautiful history of this exam, Enjoy the nostalgia by viewing the same book which they studied many years ago for preparing for this exam. Looking at the same module which they used in Kota, back in there days. Hope you understand that.
